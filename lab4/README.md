# P2P Encrypting CLI Messenger in Erlang

Простой p2p-мессенджер с шифрованием на ерланге.  
Позволяет обмениваться сообщениями напрямую без центрального сервера.  
Запускается через escript и использует TCP с ручным handshake и обменом ключами.

---

## Запуск

Собрать или просто запустить напрямую:

```bash
escript p2p_messenger.escript **port**
```

Отправка

```bash
>send localhost 5001 hello world
```

Пример получаемого сообщения

```bash
[p2p_conn] sending handshake to {{127,0,0,1},5001}
[p2p_conn] handshake send ok to {{127,0,0,1},5001}
<< From {127,0,0,1}:5001 @ at 2025-12-15T22:19:22 message: hello world
```

# Ключевые моменты реализации

## p2p_node.erl
Код ключевой ноды, сервера, который отвечает за прием соединений и управление пирами

## p2p_conn.erl
Модуль для установления соединений и обмена зашифрованными сообщениями, отвечает за handshake и очередь сообщений.

## p2p_crypto.erl
Модуль генерации ключей и шифрования

## p2p_cli.erl
Модуль CLI для взаимодействия с юзером

## p2p_history.erl
Модуль хранения истории сообщений (локально)
